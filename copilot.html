Copilot's Journey

MARCH 2023
Microsoft 365 Copilot announced for enterprise customers, bringing AI to Word, Excel, PowerPoint, Teams, and Outlook.
JUNE 2023
GitHub Copilot X announced with GPT-4. Adds AI code review, PR summaries, and documentation generation.
